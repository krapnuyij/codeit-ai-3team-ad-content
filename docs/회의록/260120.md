## 주요 안건 및 논의사항

### 1. 진행 현황 공유

- 김명환: 서빙 서버, 이미지 생성
- 김민혁: 텍스트 생성 모델
- 박지윤: 진행 사항 및 일정 체크, 데이터 조사
- 이건희: 프론트엔드/백엔드 구축
- 이솔형: 불참

### 2. 핵심 논의사항
**주요 이슈**:
- 프롬프트로 이미지 생성 실험
- 발표자료용 나노바나나 이미지 생성
- MCP 관련 코드 수정
- HTML 방식 텍스트 생성 확정
  - 텍스트 위치를 LLM에게 이미지를 보고 빈 공간에 넣어달라고 했을 때 불안정
  - 좌표를 AI 모델로 찾는 것 제안했으나 시간 부족
  - 부가 정보 텍스트 위치를 오른쪽 아래나 왼쪽 아래에 넣는 방식 고려 중
  - 텍스트 바 생성 실험 중
- 프론트엔드 입력 시 이미지 입력 없이 프롬프트로 이미지 생성하도록 프론트엔드 수정 중
- AI 모델 미확정으로 프론트엔드/백엔드에 합치는 작업 어려움
- 프론트엔드에서 점포 홈페이지 생성 실험
