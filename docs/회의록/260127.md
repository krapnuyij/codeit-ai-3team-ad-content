## 주요 안건 및 논의사항

### 1. 진행 현황 공유
**각자 진행 상황**:
- 김명환: 추가 테스트, 발표 자료 제작
- 김민혁: 텍스트 생성 모델링, 발표 자료 제작
- 박지윤: 발표 자료 총괄, 품질 평가, 진행 사항 및 일정 체크
- 이건희: 발표 자료 제작
- 이솔형: 발표 자료 제작

### 2. 핵심 논의사항
**주요 이슈**:
- FLUX.2 kein 구동 성공
  - 4비트 양자화 모델로 테스트해본 결과, 메모리 약 21.9GB를 차지하며 L4 GPU에서 구동 성공
  - 속도 향상: 기존 10~15분 걸리던 생성이 약 1~2분 (1/10 수준)으로 획기적으로 단축됨
  - 품질 확인: 기존에 실패했던 노가리 이미지도 잘 생성되고, 바나나 표면에 2500원 글자를 자연스럽게 합성하는 등 이미지 편집 성능이 뛰어남
- 회고
  - 1~2주만 더 있었으면 FLUX.2로 완벽하게 구현했을 것
  - 단순히 프롬프트 생성이 아니라 Image-to-Image 방식으로 접근했다면 상품 위에 자연스럽게 글자를 입히는 것이 가능했을 것
- 텍스트 위치 자동화 (Segmentation)
  - 사용자가 텍스트 위치를 Auto로 설정했을 때 작동하는 로직 구현
  - 개인 브랜치에 푸시
- 정성적 평가 기준표(Rubric)를 만들어서 팀원 설문
  - 발표 자료의 품질 평가 파트에 반영
- 발표 자료 작성
  - 캔바 협업을 통해 PPT 작업 동시 수행
