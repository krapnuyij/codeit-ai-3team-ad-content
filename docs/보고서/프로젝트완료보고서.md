## 1. 프로젝트 개요

### 1.1. 배경 및 목적

생성형 AI 기술을 활용하여 소상공인이 손쉽게 광고 콘텐츠를 제작할 수 있도록 지원하는 All-in-One 통합 플랫폼입니다.

### 1.2. 핵심 기능

- 광고 배너 이미지 자동 생성
- 상품 홈페이지 자동 생성
- 대화형 광고 생성
- MCP 프로토콜 지원

## 3. 기술 스택 및 아키텍처

### 3.1. 기술 스택

- FLUX.1-dev, BiRefNet, Qwen2-VL
