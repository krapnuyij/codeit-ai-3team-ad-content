## [2] 오늘 맡은 역할 및 구체적인 작업 내용
**답변**:  
```
1. Google Cloud VM 설정 지원
   - 팀원의 GCP VM 환경 구축 지원 및 설정

2. 광고 이미지 생성 파이프라인 검토
   - 이미지 생성 모델 및 파이프라인 구조 분석 진행 중

3. 바나나 이미지 생성 테스트 완료
   - 테이블 위 바나나 이미지 생성 성공 확인

4. 메모리 최적화 설정
   - 메모리 부족 이슈 해결을 위한 양자화(quantization) 적용
   - CPU 스왑 메모리 설정

5. 이미지 생성 성능 측정
   - 이미지 1장 생성 시 약 3~4분 소요 확인

6. GitHub Pages 생성 지원
   - 팀 문서화 사이트 구축 지원

7. GitHub 미러링 설정
   - 저장소 미러링 구성
```

## [5] 오늘 분석/실험 중 얻은 인사이트나 발견한 문제점은?
**답변**:  
```
[발견한 문제점]
- 이미지 생성 시 메모리 부족 이슈 발생
- 이미지 1장 생성에 3~4분 소요로 배치 처리 시 시간 최적화 필요

[해결 방안]
- 모델 양자화를 통한 메모리 사용량 절감
- CPU 스왑 메모리 설정으로 OOM 방지

[인사이트]
- 실시간 생성보다는 배치 처리 방식이 적합할 것으로 판단
- 텍스트 합성 단계를 별도 파이프라인으로 분리 필요
```

## [8] 내일 목표 / 할 일
**답변**:  
```
1. 이미지에 자연스러운 텍스트 합성 기능 구현
   - 텍스트 오버레이 방법론 검토
   - 폰트, 위치, 스타일 자동 조정 로직 설계

2. 모델 검토 완료 후 파이프라인 및 모델 관리 서버 검토
   - 모델 버전 관리 체계 설계
   - 서빙 인프라 구성 검토

3. 전체 아키텍처 설계
   - 시스템 구성 요소 정의
   - 데이터 플로우 설계

4. 전체 파이프라인 구성
   - 이미지 생성 → 텍스트 합성 → 후처리 파이프라인 설계
   - 배치 처리 및 스케줄링 방안 검토

5. 파이프라인 구간별 인터페이스 설계
   - API 스펙 정의
   - 입출력 데이터 포맷 표준화
```
