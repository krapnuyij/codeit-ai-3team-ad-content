## [5] 오늘 분석/실험 중 얻은 인사이트나 발견한 문제점은?
**답변**:  
```
1. Docker 컨테이너화 장점
   - 환경 일관성 확보로 배포 시 발생 가능한 의존성 문제 해결
   - 개발/운영 환경 분리를 통한 안정성 향상
   
2. 진행도 표시 개선 필요성
   - 기존 로직은 단순 비율 계산으로 실제 진행 상태 반영 부족
   - 작업 단계별 가중치 적용으로 정확도 향상
   
3. 문서화의 중요성
   - API 문서가 명확할수록 팀원 간 협업 효율 증가
   - 예제 코드 포함 시 이해도 크게 향상
```
